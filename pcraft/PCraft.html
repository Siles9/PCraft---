Настройка API-ключей для маркетплейсов

Wildberries

Токен можно получить в личном кабинете WB Partners → Настройки → Доступ к API

Ozon

Ключи создаются в Ozon Seller → Настройки → Интеграция → API-ключи

Яндекс.Маркет

API-Key создаётся в партнёрском интерфейсе → Настройки → Доступ к API
🔐 Ключи хранятся только в вашем браузере и никуда не передаются.
Для работы парсера необходимо ввести действующие ключи.
